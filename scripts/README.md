# Scripts

Folder ini berisi script-script untuk training model dan data processing.

## Files:

- `train_and_export.py` - Script utama untuk training model
- `prepare_dataset.py` - Prepare dan split dataset
- `test_dataset.py` - Test model dengan dataset
- `convert_to_tfjs.py` - Convert model ke TensorFlow.js
- `convert_h5_to_tfjs.py` - Alternative converter
- `save_model_only.py` - Save model weights only
- `copy_of_phishing_image_detection.py` - Backup/reference code
- `training_history.png` - Grafik hasil training

## Usage:

Lihat dokumentasi lengkap di folder `docs/`.
